# CodePlan
Code Plan ，从AI聊天式写代码，变为输入需求后有AI主导，让开发者被动做选择题的方式完善计划，拆分任务，逐个完成。开发者不再需要精通提示词及自然语言。

它完全有Codex进行制作，没有任何人为代码掺杂，新时代产品，无古老人为编码。



## 特点
* 更好的任务执行，不在需要开发者精通自然语言、提示词，它里面全部集成了！
* 无门槛， 开发者只需要输入开发计划，然后它会自检你的任务的模糊点，让你做选择题的方式完善计划，推动计划完善。
* 自动分任务分步骤，让你全流程可观测，步骤清晰，全局掌控。
* 通过一个URL通过浏览器打开直接进行使用，跨平台！
* 可电脑部署，出门后手机直接通过浏览器控制使用，多个不同的终端数据同步。
* 多计划多任务同步进行。你有灵感时，可以迅速创建几十个计划，而不必再去等待之前的codex的任务结束
* 内置同步文件锁，不用担心多个计划及多个任务同时操作一个文件导致产生脏数据，任务会自动进行等待自动分流
* 多模型验证，比如主模型用 codex ，一些验证等周边的杂活交给 DeepSeek V4 ，降低tokens消耗。
* 可以设置第三方的openai，让codex使用第三方openai工作。
* 自动接入 GiteeAI 的大模型平台算力，手机号开通后直接接入使用。
* 浏览器多终端面板，可平铺，多个终端同时进行，方便一个人开9个终端并行同步工作，而不用频繁来回切换。
